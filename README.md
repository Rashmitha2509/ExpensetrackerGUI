## Expense Tracker and Budget Manager

A Java-based desktop application developed to manage personal expenses and monitor monthly budgets. The system provides a graphical user interface for recording expenses, categorizing spending, and analyzing financial data.

## Overview

Managing day-to-day expenses manually can be inefficient and error-prone. This application addresses that problem by providing a structured and interactive way to track expenses and control spending.

The system is designed as part of a Java internship project to demonstrate practical implementation of GUI-based desktop applications using core Java concepts.

## Features

- Add and manage expense records  
- Categorize expenses (e.g., Food, Travel, Bills, Shopping)  
- View expense history in tabular format  
- Calculate total expenditure  
- Set and monitor monthly budget  
- Display warning when budget exceeds  
- Store data persistently using CSV files  
- Basic graphical representation of expenses  

## Technologies Used

- Java (Core Java)
- Java Swing (GUI Development)
- Object-Oriented Programming (OOP)
- File Handling (CSV-based storage)
- Java Collections Framework

## How to Run

1. Open the project in Visual Studio Code  
2. Open terminal in the project directory  

Compile the project: javac *.java

Run the application: java MainApp

## Usage

- Enter expense details including category, amount, date, and description  
- Set a monthly budget  
- Add expenses and monitor total spending  
- System will display an alert when the budget limit is exceeded  
- View recorded expenses and summary
  
## Objective

The objective of this project is to implement a real-world financial tracking system using Java. It demonstrates key concepts such as GUI design, event handling, file storage, and modular programming.

## Limitations

- Uses local file storage instead of database  
- Limited data visualization features  
- Single-user application
